```
#include<stdio.h>
int main()
{	
	int i;
	int j;
	int k;
	int m;
	int x;
	int temp;
	printf("请输入要输入的数字个数：");
	scanf("%d",&x);
	int array[x];
	int len = sizeof(array)/sizeof(array[0]);
	printf("done\n");
	printf("请输入%d个数：\n",x);
	
	for(m = 0;m < len;m++)
	{
		scanf("%d",&array[m]);
	}
	for(i = 0;i < len-1;i++)
	{
		for(j = 0;j < len-1-i;j++)
		{
			if(array[j]>array[j+1]) 
			{
			temp = array[j];
			array[j] = array[j+1];
			array[j+1] = temp;
			}
		}
	}
	printf("正序排列：\n");
	for(k = 0;k<=len-1;k++)
	{
		printf("%d ",array[k]);
	}
	printf("\n");
	printf("倒序排列：\n");
	for(k = len-1;k>=0;k--)
	{
		printf("%d ",array[k]);
	}
	return 0;
}
```
**核心思想：**
通过重复交换相邻的逆序元素，使最大/最小值逐渐“浮”到数组末端。

### 核心思想详解：

1. ​**相邻比较与交换**​（算法精髓）  
    依次比较相邻的两个元素：
    
    - 如果顺序错误（如 `arr[j] > arr[j+1]`），则交换它们
    - 每次交换确保较大值向数组右侧移动
2. ​**多轮冒泡过程**​
    
    - 进行 n-1 轮遍历（n 为元素数量）
    - 每轮都会使当前未排序部分的最大值"冒泡"到其最终位置  
3. ​**逐渐收缩范围**​  
    每轮结束后，排序范围自动缩小：
    - 第1轮：比较范围 [0, n-1]
    - 第2轮：比较范围 [0, n-2]（末尾元素已有序）
    - ...
    - 第i轮：比较范围 [0, n-i-1]
4. ​**提前终止优化**​  
    增加交换状态标记：
    - 若某轮未发生任何交换 → 数组已有序 → 立即结束排序
