# 一维数组
1. 数组的定义及表示
```
#include<stdio.h>
int main()
{
//数组定义
	int array0[10];//定义一个数组：数据类型+变量名[数组长度]
	int array1[3] = {1,2,3};//将数据按顺序放在一个花括号内，数据间用逗号分割。
	int array1[10] = {1,2,3};//倘若只写了前几位，那么后面剩下的自动写入0，数组长度不变。
	
	int array2[10] = {0};//将所有元素初始化为0.
	int array3[] = {1,2,3}//此写法是根据花括号里有多少元素自动设定数据长度

array[0] = 3;//用下标法单独访问单独数组元素，数组下标从0开始，数组地址连续


//用for循环写入数组数据
	for(data = 0;data<10;data++)
	{
		array[data] = data+100;
	}
//计算数组长度
int size;
size = sizeof(array)/sizeof(array[0])//sizeof()表示括号里整个数组大小，用整个数组大小除以单个数组元素大小就得到了数组的元素个数。

return 0;
}
```
